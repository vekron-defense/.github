# `.github` — VEKRON organization profile

This repository hosts the **public profile content** for the
`vekron-defense` GitHub organization.

The file at `profile/README.md` is rendered as the organization's
overview page at <https://github.com/vekron-defense>.

This repository contains:

- `profile/README.md` — public-facing organization overview.

This repository contains NO operational source code, NO architecture,
NO threat models, NO schemas, and NO infrastructure-as-code. All
operational repositories under `vekron-defense` are private and
remain so.

For inquiries, please visit [vekron.ai](https://vekron.ai).
